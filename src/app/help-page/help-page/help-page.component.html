Welcome to the Language Learning Game Help Page!

Getting Started

To begin playing, you first need to add categories which can include various topics like 'Animals', 'Foods', 'Verbs', etc. Each category should contain words related to the topic.

Playing the Games

Choose a game from the main dashboard. Each game has different rules and objectives focused on enhancing your English vocabulary:

  • Word Sorter: Determine if the given word fits the category shown.
  • Mixed Game: Unscramble the letters to form the correct word.

Adding Categories

Go to the 'Manage Categories' section from the main menu where you can add or remove categories and their corresponding words.

Need More Help?

If you encounter any issues or have questions, feel free to contact our support team.
